# Cluster Topology

## Overview

This document outlines the multi-cluster setup strategy for development, staging, and production environments.

## Cluster Design

### Development Cluster
- Single-node or minimal setup
- Rapid iteration environment
- Namespace-per-developer model

### Staging Cluster
- Production-like configuration
- Integration testing
- Performance validation

### Production Cluster
- High availability setup
- Multi-zone deployment
- Strict access controls
